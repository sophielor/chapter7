HTML: Lists



In this chapter we will be talking about what is < b > HTML provides three ways to specify lists: ordered lists, unordered lists, and description lists. Ordered lists use numbers or letters to indicate the "order of list elements, unordered lists use symbols such as bullets to list elements in no particular order, and description lists use indents to list elements along with their descriptions.

Ordered Lists
- An ordered list can be created with the < ol > tag and each list item can be created with the < li > tag.
- In MANUALLY CHANGING THE NUMBERS, The start attribute can be used to manually set the starting number of an ordered list in HTML, and the list will continue to increment by one as usual.
- You can also explicitly set a certain list item to a specific number.
- The reverse attribute in HTML can be used to display ordered list items in reverse order. This can be useful when adding new items to a list, such as podcast episodes or presentations, and wanting the most recent items to appear first.

Unordered Lists
- This will produce a bulleted list.

Use < ol > when the list items have been intentionally ordered, and the order should be emphasized. Use < ul > when changing the order of the items does not make the list incorrect.

Nested Lists
- You can nest lists to represent sub-items of a list item.
- The nested list has to be a child of the li element. You can nest different types of list.

Description Lists
- A description list can be created with the dl element. It consists of name-value groups, where the name is given in the dt element and the value is given in the dd element.
- A name-value group can have more than one name and/or more than one value.
